# 🩺 Identifying Diabetes Detection Using Machine Learning Algorithms with Kernel PCA

## 📌 Overview

This project focuses on the early detection of diabetes using Machine Learning algorithms and Kernel Principal Component Analysis (Kernel PCA). The system analyzes patient health records and predicts whether a person is diabetic or non-diabetic.

A total of **15 Machine Learning classification algorithms** were implemented and compared using different train-test split configurations and evaluation metrics.

Among all models, the **Kernel Support Vector Machine (KSVM)** achieved the highest performance with **100% accuracy** using the 80-20 train-test split.

---

# 🎯 Objectives

* Detect diabetes at an early stage using machine learning.
* Compare multiple classification algorithms.
* Improve prediction accuracy using Kernel PCA.
* Reduce complexity through dimensionality reduction.
* Assist healthcare professionals with intelligent prediction systems.

* **Total Records:** 521 patient records

* **Target Classes:**

  * `0` → Non-Diabetic

# 🧠 Proposed Methodology

## Step 1: Data Collection

Patient health records are collected and organized into a structured dataset.

The dataset is cleaned and prepared using:

* Feature scaling

## Step 3: Dimensionality Reduction

# 📈 Performance Metrics

The following metrics were used to evaluate the models:

## Accuracy

Measures the percentage of correct predictions.

## Precision

Measures how many predicted positive cases are actually correct.

## Recall

Measures how many actual positive cases are correctly identified.

## F1-Score

Harmonic mean of precision and recall.
